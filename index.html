The Kit

This kit is designed to foster creative solutions to problems big and small.

The Kit includes:

  • Arduino Uno
  • Basic inputs such as buttons and potentiometers
  • Prototyping Boards
  • LEDs, and LCDs
  • Servos, Steppers, and DC Motors (oh my!)
  • And much more!

Download Bill of Materials

About

The Global Maker Initiative (GMI) was created to encourage the development of solutions to problems in areas with extreme poverty. GMI aims to do this through giving projects that can be made with the kit.

Whether you are a student, or a professional, you can contribute to this project by sending your ideas/product instructions to globalmakerinitiative@gmail.com to be published on this site.

UV Water Purifier

This project harnesses the bacteria killing properties of ultraviolet waves to purify water. This project utilizes the timer on Arduino and a potentiometer which the user adjusts for the volume of water. After the purifier has been in the water for enough time, the indicator light will light up, and the UV LED will power off.

Hardware for this project:

Arduino Uno
Colored LED
Ultraviolet LED (wavelength of 328nm to 210nm)

Wiring Diagram:

The code is available on GitHub. Use PontentiometerBasedPurifier.ino under "PotetiometerBasedPurfier"

The STL Model for this project can be found on Thingiverse

  • Code: GitHub Repository
  • Author: Becky Button

Aid for Blind

This device works by taking the values from the ultrasonic, and scaling them to the range of a servo. The user holds the device, and can tell how far something is by the angle the servo is at.

Hardware for this project:

Arduino Uno
9g Servo
HC-SR04 Ultrasonic

Wiring Diagram:

The code is available on GitHub.
